هذا المشروع يهدف إلى تطوير نظام إدارة كلمات المرور باستخدام ASP.NET Core Web API مع Entity Framework Core وقاعدة بيانات SQL Server. يتيح هذا النظام إنشاء حسابات جديدة وتسجيل الدخول، بالإضافة إلى إمكانية استعادة كلمات المرور المنسية وتغييرها.

الميزات الرئيسية:
إنشاء حساب جديد:
يسمح للمستخدمين بإنشاء حسابات جديدة عبر API مخصصة لإنشاء المستخدمين الجدد.
تسجيل الدخول:
يتيح للمستخدمين تسجيل الدخول إلى حساباتهم باستخدام API مخصصة للتحقق من صحة اسم المستخدم وكلمة المرور.
نسيان كلمة المرور:
يوفر API لاستعادة كلمة المرور المنسية عبر إرسال بريد إلكتروني يحتوي على رابط لإعادة تعيين كلمة المرور.
إعادة تعيين كلمة المرور:
يسمح للمستخدمين بتغيير كلمات المرور الحالية باستخدام API مخصصة لتحديث كلمات المرور.
العمليات الأساسية:
إنشاء حساب جديد:
يستخدم API POST لإضافة مستخدم جديد إلى قاعدة البيانات.
تسجيل الدخول:
يستخدم API POST للتحقق من صحة اسم المستخدم وكلمة المرور وإصدار رمز مصادقة (Token) في حالة النجاح.
نسيان كلمة المرور:
يستخدم API POST لإرسال بريد إلكتروني يحتوي على رابط لإعادة تعيين كلمة المرور.
إعادة تعيين كلمة المرور:
يستخدم API PUT لتحديث كلمة المرور للمستخدم المستهدف بعد التحقق من صحة الرابط المُرسل لإعادة تعيين كلمة المرور.
تقنيات العمل:
ASP.NET Core Web API:
استخدام ASP.NET Core لتقديم واجهات برمجة التطبيقات (API) للتفاعل مع التطبيق عبر HTTP.
Entity Framework Core:
استخدام Entity Framework Core للتفاعل مع قاعدة بيانات SQL Server بشكل مباشر وتنفيذ عمليات إدراج واستعلام وتحديث البيانات.
SQL Server:
استخدام قاعدة بيانات SQL Server لتخزين بيانات المستخدمين وكلمات المرور وغيرها من المعلومات ذات الصلة.
الملاحظات الإضافية:
يجب تأمين جميع نقاط الاتصال وعمليات تغيير كلمات المرور بشكل آمن باستخدام بروتوكول HTTPS وتقنيات التشفير المناسبة.
ينبغي تنفيذ ميزات إرسال البريد الإلكتروني باستخدام خدمات البريد الإلكتروني المناسبة (مثل SMTP) لإرسال روابط إعادة تعيين كلمات المرور بشكل فعال وآمن.
